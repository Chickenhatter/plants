<h1>This was made for a hackclub event</h1>
<p>This was made for horizons</p>
<p>This was made with no AI</p>
<h3>Movement</h3>
<p>Use WASD for movement, use m to feed plants</p>
<p>Press k to pick up nitrogen/water and i to drop them</p>
<p>Just run into the compost/tap to get more plant food</p>
<p>Plant heath cannot be recovered</p>

<p>For this game you just play until you loose and fail to keep all plants alive</p>
<p>This game is called gardening for coders</p>
<p>One issue you may encounter which is hard to pull off is that eventually you would theoretically see the bar of the plant go below the pot but this is theoretical</p>
<p>You can play this on itch.io</p>
